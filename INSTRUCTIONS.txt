## Installation Instructions ##

import class, images, and lib folders into \Documents\AutoHotkey\ 
import master_switch file into \Documents\AutoHotkey\ 
install SciTE4AutoHotkey

In scite4autohotkey open master_switch and set file path for class and gui

CREATE MASTER SWITCH SHORTCUT with RUN AS ADMINISTRATOR
In iCUE software import Corsair Profiles: "keyboard" and "ESPRIT 1"
Make sure all settings are checked yes, set "keyboard" as default
